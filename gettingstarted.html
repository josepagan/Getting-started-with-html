Getting Started with HTML notes

The world wide web

The world wide web is composed by HTML documents. These reside in servers.
Clients access servers through the internet using a special protocol (http)

HTML as computer language

HTML is a computer language and require precision rather than ambiguity.
A HTML like any other computer language has to follow syntax and structure rules so it can be readed by the computer
A HTML document should also be easy to read, a proper code structure must be applied to make the code accesible for future needs

The HTML document

A HTML document may contain basic text content, which is displayed directly, then mark up code, which will give form to the text. It may contain references to other documents such as images and videos and it might contain links to other HTML documents.

TAGS

Tags describe HTML documents, they either dictate how the content is presentend or form the structure
In HTML tags follow this structure:
<TAGNAME>CONTENT</TAGNAME>, although void tags are not closed
Tags may need attributes, attributes contain values required by the tag
Some tags operate directly inline content, some others generate elements (blocks) that will include content

SOME TAGS REVIEW

<b></b> Makes enclosed content bold
<em></em> Makes enclosed content italic
<a href=""></a> Create links to other html, requires attribute href with a value (the url of the linked page)
<img src="url" alt="text"> Serve images, src attribute contains the reference to the image, alt attribute contains information either alternative or complementary to the image
<br> Breaks the line content
into a different lines
<p></p>

generates a paragraph block

HTML STRUCTURE

Some tags are structural. They generate the structure of the document, they wrap the content, but it does not affect it like mark up tags.
A streamlined HTML document follows this structure:

<!DOCTYPE HTML> marks document as HTML
<HTML>(surrounds document)
<HEAD> (contains metadata and title)
<BODY> (contains content)
</BODY>
</HEAD>
</HTML>
